# sensemaking skill
A Codex skill that helps clarify messy questions into what to ask, what to watch, and what to do next.

# 把事情看明白skill介绍

`sensemaking` 是一个 Codex skill，用来帮助人在信息混乱时，寻找“可判断的确定性”。

它不是直接替你下结论，而是帮你把复杂问题整理成三个问题：

1. 我到底在问什么？
2. 我应该看什么？
3. 我下一步怎么办？

## 适合什么问题

适合那些没有标准答案、但会影响判断和行动的问题。

例如：

- 2026 年我该不该更关注股票资产？
- 房地产是不是已经到底？
- 我要不要换行业？
- 这个政策有没有用？
- 这本书里的观点靠谱吗？
- 这件事对我意味着什么？

## 核心思路

当信息很多、观点很多、脑子很乱时，不要急着下结论。

先问：

```text
我真正想判断的是什么？
哪些条件或信号最关键？
什么情况说明我判断错了？
我现在能做的下一步是什么？
```

## 适用场景

- 读书和学习
- 投资和资产配置
- 职业选择
- 城市和买房决策
- 政策和新闻解读
- 复杂人生选择

## 不适合什么问题

这个 skill 不适合简单事实查询、词语定义、常规代码报错等问题。

它更适合这类问题：

```text
该不该？
会不会？
有没有机会？
靠不靠谱？
我该怎么看？
这对我意味着什么？
```

## 一行命令安装

```bash
npx skills add miky-air/sensemaking -g
```

安装完成后，重启 Codex，让它识别这个 skill。

## 使用示例

```text
用 sensemaking 分析：房地产是不是到底了
用 sensemaking 分析：我该不该换行业
用 sensemaking 分析：这本书的观点靠不靠谱
```
